Learning Git and GitHub

THE PULL COMMAND

  • Just modify the code in git directly
  • Now to pull the change to vscode
  • COMMAND: git pull

Basic Git Commands:

git init To connect the folder to the git repo
git add file_name // git add . to add the file to the staging area
git commit -m "MESSAGE" To convert object / to get file ready to be pushed
git push -u origin BRANCH_NAME // git push To push the file into the global repo
git stash To pause or revert back changes before push(Refer https://git-scm.com/docs/git-stash)

Learning Git and GitHub

THE PULL COMMAND

  • Just modify the code in git directly
  • Now to pull the change to vscode
  • COMMAND: git pull

Basic Git Commands:

git init To connect the folder to the git repo
git add file_name to add the particular file to the staging area
git add . To add wholem folder to staging area
git commit -m "MESSAGE" To convert object or to get file ready to be pushed -m details of the message to be conveyed
git push -u origin BRANCH_NAME To push the particular file into the global repo
git push To push whole folder into global repo
git stash To pause or revert back changes before push(Refer https://git-scm.com/docs/git-stash)
